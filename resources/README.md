# Resources (kept minimal)

## Core (the plan's picks)
- **ML theory (applied):** Géron, *Hands-On Machine Learning* (3rd ed.) — primary applied depth.
- **ML theory (statistical):** *An Introduction to Statistical Learning* (ISLR) — free PDF; statistical grounding.
- **DSA:** NeetCode pattern roadmap + my LeetCode Easy collection.
- **System design:** Chip Huyen, *Designing Machine Learning Systems* — framework + cases.

## Supplementary (optional, don't over-collect)
- *The Elements of Statistical Learning* (ESL) — deeper math, reference only.
- 3Blue1Brown neural network series — backprop intuition (Week 7).
- The Illustrated Transformer (Jay Alammar) — Week 8.
- StatQuest (Josh Starmer) — quick intuition refreshers.
